# Instructor Materials

## Overview

This directory contains comprehensive instructor materials for the "Building Agentic AI Applications with LLMs" course. These materials support instructors in delivering high-quality instruction that prepares students for the NVIDIA-Certified Professional: Agentic AI (NCP-AAI) certification exam.

## Quick Start Guide

### Before Course Starts

1. **Read**: `instructor_guide.md` (overview and Module 1)
2. **Review**: `teaching_guidance.md` (teaching philosophy)
3. **Prepare**: Module 1 materials and lab environment
4. **Test**: All technical setup (API keys, platforms, labs)

### Before Each Module

1. **Review**: Module lesson plan in `instructor_guide.md`
2. **Prepare**: Demonstrations and examples
3. **Check**: Lab solution in `lab_solutions_guide.md`
4. **Anticipate**: Common questions from `faq.md`

### During Labs

1. **Reference**: Lab solutions guide for debugging
2. **Use**: FAQ for common issues
3. **Apply**: Teaching guidance for support strategies

### During Grading

1. **Use**: Appropriate rubric from `grading_rubrics.md`
2. **Apply**: Grading guidelines consistently
3. **Provide**: Specific, actionable feedback

### For Continuous Improvement

1. **Reflect**: After each class and module
2. **Gather**: Student feedback regularly
3. **Adjust**: Based on student performance and feedback
4. **Update**: Materials as needed

- Review materials thoroughly before teaching
- Test all code examples and labs
- Prepare backup plans for technical issues
